# Yoga Pose Classifier

Веб-застосунок для автоматичного розпізнавання йога-асан з відео за допомогою штучного інтелекту. Користувач завантажує відео своєї практики, формує план тренування з випадаючого списку, а система порівнює виконані асани з планом та зберігає результати в історії.

## Функціонал
- Завантаження відео йога-практики (mp4, mov тощо)
- Формування плану тренування (вибір асан з випадаючого списку)
- Автоматичне розпізнавання асан кожні 5 кадрів
- Порівняння виконаних асан з планом
- Збереження історії тренувань (дата + виконані пози + тривалість)

## Технології
- Python
- Gradio (веб-інтерфейс)
- EfficientNetB0 (Keras/TensorFlow) — модель класифікації
- OpenCV — обробка відео
- NumPy — допоміжні обчислення

## Як запустити локально
1. Клонуйте репозиторій : ash
   git clone https://github.com/7Dan1el7/yoga-pose-classifier.git
   cd yoga-pose-classifier
2. Встановіть залежності: pip install -r requirements.txt
3. Запустіть застосунок: python app.py

## Структура репозиторію

- app.py — основний код застосунку
- requirements.txt — залежності
- yoga_history.txt — (генерується автоматично) історія тренувань
- efficientnetb0_yoga_model.keras — модель (посилання для завантаження: https://drive.google.com/file/d/1SJLYtf0UT0KOgnyUvFgfTzNNcul3Ug62/view?usp=sharing)
